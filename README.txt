Как пользоваться
================

1. Сейчас сайт доступен по временной публичной ссылке:

   https://d721097386892b.lhr.life

   QR-код qr-code.png уже ведет на эту ссылку.

2. Важно: текущая публичная ссылка работает, пока на этом компьютере запущены
   локальный сервер и SSH-туннель localhost.run. Для настоящего подарка лучше
   выложить всю папку love-qr-site на постоянный хостинг.
   Быстрые варианты: Netlify Drop, Vercel, GitHub Pages, любой обычный static hosting.

3. Когда получишь постоянную публичную ссылку на сайт, пересобери QR-код:

   python make_qr.py "https://твоя-публичная-ссылка"

4. Готовый QR появится в файле qr-code.png.

Где менять текст
================

- Главная фраза: index.html, блок <h1>
- Текст под фразой: index.html, класс message__text
- Текст после кнопки: index.html, блок id="secret"

Важно
=====

Текущий qr-code.png ведет на временную публичную ссылку localhost.run.
Она не вечная. Если компьютер выключить, закрыть сервер или туннель, ссылка перестанет работать.
